Múltiples Penalizaciones

Cómo agregar varias penalizaciones a la vez
Puedes agregar múltiples penalizaciones en el campo de razón separándolas con un punto (.)

Ejemplo:

Se ha estrellado contra otro piloto. Se ha saltado una curva.

 Tip: Cada frase separada por punto se considerará una penalización independiente para el mismo piloto.
